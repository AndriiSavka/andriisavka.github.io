Command Line

Ctrl+Shift+T – open new terminal windows in tab;

touch index.html – create file;

Multi copy:

for ex:
cp /home/usr/dir/{file1,file2,file3,file4} /home/usr/destination/
or
cp /home/usr/dir/file{1..4} home/usr/destination/
if you in dir from you will copy:
cp file{1..4} /home/usr/destination/

ls -ln /var/www/html - display and sort files in derictory html;

mkdir folder – create folder;

history – history of commands;

!! - запуск попередньої команди;

cp -R ../Downloads/* . - скопіювати рекурсивно всі файли з каталогу Downloads в текущий каталог;

Ctrl+C – відмінити запуск команди;

| - передати результат виконання однієї команти в другу наприклад:
ls -ln | grep favicon - виведи список і вибери все що містить слово favicon

ls -ln > output.txt – записати виведене в файл;

cat output.txt – вивести на екран вміст файлу;

mv /home/jack/testfile /home/jack/testfile2 - move file;

cat output.txt | grep favicon - output text and grep favicon from it;
