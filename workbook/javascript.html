Змінні

var - оголошення змінної;
for example: var myMessage;

myMessage = "Hello, User!"; - записти дані в змінну;

var myMesage;
myMessage = "Hello, User!";

myMessage = "Hello, User!";

myMessage = "Bye, User!"; - дані можна перезаписувати;
alert(myMessage);

Для назв змінних існує 3 обмеження:

  1. Ім’я може складатись з:
    • букв ;
    • цифр ;
    • символів $ ;
    • _ ;
  2. Назви змінних не можуть починатись з цифри;
  3. Не можна використовувати в якості назви змінної зарезервовані слова;

Правила вибору назви змінної

  1. Ніякого трансліту, тільки англійська;
  2. Короткі імена можна давати тільки змінним, які не несуть смислового навантаження, наприклад індексам;
  3. Змінні, що складаються з кількох слів, пишуться в camelCase;
  4. Ім’я змінної має чітко відповідати її змісту.

Типи даних

  1. Число ‹‹ number ›› ;

  3. Логічний тип (булевий) - Boolean ;
  4. Спеціальне значення ‹‹ null ›› ;
  5. Спеціальне значення ‹‹ undefine ›› ;

Арифметика та присвоєння. Основні математичні оператори Javascript

  1. Стандартні оператори: "+", "-", "/", "*"

    var a = 5;
    var b = 6;

    console.log(a + b);

        Also

    var f = 'my' + 'String';

    console.log(f);

        Also

    console.log('1' + 1); - буде 11 ;

  2. Інкремент: "++" - збільшення змінної на 1;

    var i = 2;
    i++;

    console.log(i);

  3. Декремент: "--" - зменшення змінної на 1;

    var i = 2;
    i--;

    console.log(i);

    можна застосовувати тільки до змінних:
    console.log(5++) - так не працює

  4. Оператори порівняння: "<" ">", "<=" ">=", "==", "==="

    = - означає присвоїти;
    == - означає перевірити рівність;
    === - перевірка не тільки на рівні значень, але й на рівність типів;

  5. "Не рівно" - "!=" або "!=="

    console.log('01' !== 1);
    console.log('01' != 1);

Взаємодія з користувачем

  1. Alert => alert('Message');

  2. Prompt => var result = prompt(title, default);

    Виводить модальне вікно prompt із заголовком title і полем для вводу тексту(можна підставляти дефолтний набір тексту замість default), кнопку ok та cancel;
    Prompt повертає те, що ввів користувач і присвоєю змінній result (якщо ok,а якщо cancel - то null);

    Наприклад:
    var name = prompt('What is your name?', 'Jo')

    alert('You name is:' + name);

    P.S. +prompt - означає приведення до числа;

  3. Confirm => var result = confirm(question);

    Виводить модальне вікно з запитанням question і 2-ма кнопками - OK & CANCEL;
    Змінній result присвоїться true, якщо користувач натисне OK та false, якщо CANCEL.

    var isAdult = confirm('Are you adult?');

    alert('is adult = ' + isAdult);

Цикл - багаторазове використання коду

Повторення циклу називається ітерацією

Види циклі:

  1. while

    while (condition) {
       //
    }

    Наприклад:

    var i = 0;
    while (i < 3) {
       alert(i);
       i++;
    }

  2. do while

    do {
       // } while (condition);

    Наприклад:

    var i = 0;

    do {
       alert(i);
       i++;
    } while (i < 3);

  3. Цикл for

    for (beginning; condition; step) {
       // code
    }

    Наприклад:

    for (var i = 0; i < 3; i++) {
       alert(i);
    }

    Наприклад 2:

    var i = 0;
    for (; i < 3; i++) {
       alert(i);
    }
